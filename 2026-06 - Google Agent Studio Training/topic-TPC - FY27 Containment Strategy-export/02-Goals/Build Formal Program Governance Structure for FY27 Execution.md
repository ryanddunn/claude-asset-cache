## Success Criteria

- A formal program governance structure is in place for FY27 containment execution: defined workstreams, owners, milestones, and a steering mechanism
- Governance addresses the lack of operational process maturity in cross-functional partners — it cannot rely on ad hoc coordination
- Program status is visible in a single source of truth (the Strategy Critical Path and related artifacts) that is kept current
- Escalation path from workstream-level blockers to executive-level decisions is clear and has been used at least once
- Jared (or equivalent PM) has formalized intake processes and project plans for all six workstreams
