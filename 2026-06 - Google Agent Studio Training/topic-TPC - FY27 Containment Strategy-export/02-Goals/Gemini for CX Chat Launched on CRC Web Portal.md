## Success Criteria

- Gemini for CX chat is live and accessible to customers through the CRC web portal
- Chat is built on Agent Studio and grounded by the Knowledge Catalog
- Containment rate for chat interactions is measurable and being tracked
- Escalation path to human support is functioning correctly for interactions the agent cannot resolve
- Launch has been validated through a controlled rollout or beta before full production release
